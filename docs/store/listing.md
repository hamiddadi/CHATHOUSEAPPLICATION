- Unrestricted web access: **No**
- User-generated content / user communication: **Yes** (this triggers the UGC

- User-generated content shared: **Yes**
- Shares user's location: **Yes**
- Digital purchases: **Yes** (tips / premium)
- Expected IARC result: **Teen** (may vary by region/board).
